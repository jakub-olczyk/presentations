What is this "(a)symmetric cipher"?

Symmetric
  • same key for ENCRYPTION and DECRYPTION
  • usually quite fast
Asymmetric
  • uses 2 keys to encrypt and decrypt MESSAGE
  • one key is known to everybody = public key
    used to ENCRYPT MESSAGE to somebody
  • the other is kept hidden = private key
    used to DECRYPT MESSAGE to you
